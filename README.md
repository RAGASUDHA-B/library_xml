# Library Management System (XML + JSP)

This is a simple Java web application for library management, built using JSP, Servlets, and XML data storage.

## Project Overview

- Java Maven web project
- JSP pages for user interaction
- XML file used to store book data
- Servlet-based request handling

## Features

- Add new books
- Display book list
- Search and manage book data via JSP pages

## Tech Stack

- Java
- JSP / Servlets
- XML
- Maven

## How to Run

1. Import the project into an IDE such as Eclipse or IntelliJ IDEA.
2. Build the project with Maven.
3. Deploy on a local Tomcat server.
4. Access the web app through the browser.
